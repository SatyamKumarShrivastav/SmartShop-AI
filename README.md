# SmartShop-AI
SmartShop AI is an intelligent e-commerce system offering personalized product recommendations, 6-month sales trend prediction, smart dashboards, and seamless user experience using ML and full-stack development. Built for smarter, data-driven online shopping.


# 🧠 **SmartShop AI**

### *Personalized Product Recommendation with Sales Trend Prediction*

SmartShop AI is an **AI-powered e-commerce intelligence system** that delivers personalized product recommendations, 6-month sales trend forecasting, dynamic smart cataloging, and interactive dashboards for both users and admins.
It is designed to make online shopping **smarter, faster, and data-driven** using Machine Learning and full-stack development.

---

## 🚀 **Features**

### 🛍️ Personalized Recommendation System

* Suggests products based on user interests, behavior & budget
* Uses ML models for precise and relevant recommendations

### 📈 Sales Trend Prediction (Next 6 Months)

* ML forecasting models to predict product-wise revenue
* Helps businesses plan inventory & improve decision-making

### 📊 User & Admin Dashboards

* Visual insights into sales, trends, orders & revenue
* Admin can manage products, track analytics & monitor growth

### 🚀 Smart Product Catalog

* Dynamic catalog adapts to each user’s preferences
* Enhanced shopping experience with intelligent filtering

### 💳 UPI Checkout Simulation

* Smooth, secure, and realistic payment workflow

### 🗄️ Scalable Backend & Database

* Efficient data handling
* Fast response time and secure endpoints

## 🔍 **How the AI Works**

### ✔ Recommendation Engine

* Uses collaborative + content-based filtering
* Matches users with products based on interests & buying patterns

### ✔ Sales Prediction Model

* Regression-based model trained on historical sales
* Predicts next 6-month revenue trends
* Helps in forecasting & inventory optimization

## 🧪 **How to Run the Project**

### 1️⃣ Clone the Repository

```
git clone https://github.com/yourusername/SmartShopAI.git
cd SmartShopAI
```

### 2️⃣ Install Backend Dependencies

```
cd backend
npm install
```

### 3️⃣ Start Backend

```
npm start
```

### 4️⃣ Install Frontend Dependencies

```
cd ../frontend
npm install
```

### 5️⃣ Start Frontend

```
npm start
```

### 6️⃣ Run ML Models

```
cd ../ml-models
python model_train.py
```
